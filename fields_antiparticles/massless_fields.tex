\section{零质量粒子因果场}\label{sec:massless_fields}
零质量粒子场是受限于物理的约束的，所以开始构造最一般零质量粒子场前，我们先对零质量粒子的小群 $ISO(2)$ 进行分析，看看它的生成元在 $(A,B)$ 表示中的分解情况。
\topictitle{$ISO(2)$ 生成元在 $(A,B)$ 表示中的分解}
$ISO(2)$ 小群（标准动量 $k^\mu=(\kappa,0,0,\kappa)$）的三个生成元为

\begin{equation}
    J_3= \mathcal{A}_3 + \mathcal{B}_3\,,\qquad \Pi_1 = K_1 - J_2\,,\qquad \Pi_2 = K_2 + J_1
\end{equation}
在前文的 Wigner 小群分析中，为了排除违背物理现实的“连续自旋”，我们必须强制要求物理态在“平移”生成元下特征值为零，也就是要求方程 
\begin{subequations}
    \begin{align}
        \Pi_1 \, \Ket{k,\lambda} &= 0 \\
        \Pi_2 \, \Ket{k,\lambda} &= 0
    \end{align}
\end{subequations}
不过，这两个式子解不出有效的信息；尝试构造 $\Pi_1 \pm i\Pi_2$ 的线性组合，看看能否得到更有用的约束。
在 $(A,B)$ 表示中，利用 Lorentz 群生成元的分解 $\mathbf{J} = \mathbf{A} + \mathbf{B}$，$\boldsymbol{k} = -i\mathbf{A} + i\mathbf{B}$，以及升降算符 $A_\pm = \mathcal{A}_1 \pm i\mathcal{A}_2$ 与 $B_\pm = \mathcal{B}_1 \pm i\mathcal{B}_2$
\begin{itemize}
    \item 计算 $\Pi_1 + i\Pi_2$ 的详细展开：
    \begin{align}
        \Pi_1 + i\Pi_2 &= (K_1 - J_2) + i(K_2 + J_1) \notag \\
        &= (K_1 + iK_2) + (iJ_1 - J_2) \notag \\
        \text{（代入生成元分解）} &= \bigl[ (-i\mathcal{A}_1 + i\mathcal{B}_1) + i(-i\mathcal{A}_2 + i\mathcal{B}_2) \bigr] + \bigl[ i(\mathcal{A}_1 + \mathcal{B}_1) - (\mathcal{A}_2 + \mathcal{B}_2) \bigr] \notag \\
        &= \bigl( -i\mathcal{A}_1 + \mathcal{A}_2 + i\mathcal{B}_1 - \mathcal{B}_2 \bigr) + \bigl( i\mathcal{A}_1 - \mathcal{A}_2 + i\mathcal{B}_1 - \mathcal{B}_2 \bigr) \notag \\
        \text{（重新分组）} &= \underbrace{\bigl( -i\mathcal{A}_1 + \mathcal{A}_2 + i\mathcal{A}_1 - \mathcal{A}_2 \bigr)}_{=\, 0 \text{ （}A\text{ 部分完全抵消）}} + \underbrace{\bigl( i\mathcal{B}_1 - \mathcal{B}_2 + i\mathcal{B}_1 - \mathcal{B}_2 \bigr)}_{=\, 2i\mathcal{B}_1 - 2\mathcal{B}_2} \notag \\
        &= 2i \bigl( \mathcal{B}_1 + i\mathcal{B}_2 \bigr) \notag \\
        &= 2i \mathcal{B}_+
    \end{align}
    \item 计算 $\Pi_1 - i\Pi_2$ 的详细展开：   
    \begin{align}
        \Pi_1 - i\Pi_2 &= (K_1 - J_2) - i(K_2 + J_1) \notag \\
        &= (K_1 - iK_2) + (-iJ_1 - J_2) \notag \\
        \text{（代入生成元分解）} &= \bigl[ (-i\mathcal{A}_1 + i\mathcal{B}_1) - i(-i\mathcal{A}_2 + i\mathcal{B}_2) \bigr] + \bigl[ -i(\mathcal{A}_1 + \mathcal{B}_1) - (\mathcal{A}_2 + \mathcal{B}_2) \bigr] \notag \\
        &= \bigl( -i\mathcal{A}_1 - \mathcal{A}_2 + i\mathcal{B}_1 + \mathcal{B}_2 \bigr) + \bigl( -i\mathcal{A}_1 - \mathcal{A}_2 - i\mathcal{B}_1 - \mathcal{B}_2 \bigr) \notag \\
        \text{（重新分组）} &= \underbrace{\bigl( -i\mathcal{A}_1 - \mathcal{A}_2 - i\mathcal{A}_1 - \mathcal{A}_2 \bigr)}_{=\, -2i\mathcal{A}_1 - 2\mathcal{A}_2} + \underbrace{\bigl( i\mathcal{B}_1 + \mathcal{B}_2 - i\mathcal{B}_1 - \mathcal{B}_2 \bigr)}_{=\, 0 \text{ （}B\text{ 部分完全抵消）}} \notag \\
        &= -2i \bigl( \mathcal{A}_1 - i\mathcal{A}_2 \bigr) \notag \\
        &= -2i \mathcal{A}_-
\end{align}
\end{itemize}
得出 $ISO(2)$ 的“平移”生成元在 $(A,B)$ 表示中完全分解为：
\begin{equation}
 \ \Pi_1 + i\Pi_2 = 2i \mathcal{B}_+, \qquad \Pi_1 - i\Pi_2 = -2i \mathcal{A}_- \ 
\end{equation}
为了排除物理上未观测到的“连续自旋”态，我们强制要求物理态在“平移”生成元下的特征值为零。也就是：
\begin{equation}
    \Pi_1 \, \Ket{k,\lambda} = 0, \qquad \Pi_2 \,\Ket{k,\lambda} = 0
\end{equation}
这等等价于 $(\Pi_1 \pm i\Pi_2) \ket{k,\lambda} = 0$。代入我们在上一小节得到的代数分解式 $\Pi_1 + i\Pi_2 = 2i \mathcal{B}_+$ 与 $\Pi_1 - i\Pi_2 = -2i \mathcal{A}_-$，得到了对交织方程的极端强约束：
\begin{subequations}\label{eq:ISO2-constraint-general}
    \begin{align}
    \mathcal{B}_+ \, \ket{k,a,b} &= 0 \quad \implies \quad  \ket{k,a,b}\text{ 必须是 } B \text{ 空间的最高权态，即 :} b = +B \\
    \mathcal{A}_- \, \ket{k,a,b} &= 0 \quad \implies \quad\ket{k,a,b} \text{ 必须是 } A \text{ 空间的最低权态，即 :} a = -A
\end{align}
\end{subequations}
也就是说，此刻的本征态 $\ket{k,a,b}$ 中的 $a$ 和 $b$ 已经被完全固定为 $a = -A$ 和 $b = +B$。现在只剩下最后一个生成元 $J_3 = \mathcal{A}_3 + \mathcal{B}_3$ 没有被约束了；所以有本征关系：
\begin{equation}
    J_3 \, \ket{k,-A,+B } = (\mathcal{A}_3 + \mathcal{B}_3) \, \ket{k,-A,+B} = (-A + B) \, \ket{k,-A,+B}
\end{equation}
这就得出了由 $(A, B)$ 场湮灭的无质量粒子的确定螺旋度：
\begin{equation}\label{helcity_o}
    \lambda = B - A
\end{equation}
\hrule\vspace{3ex}
非物理带来的约束问题已经讨论完毕，现在开始构造最一般的场，
在5.3节已经确定系数了$\xi$和$\lambda$以及得到了自旋统计定理，在这里我们丢掉系数；零质量对应Helcity 所以替换$\sigma$为$\lambda$,所以一般形式场可以写为：
\begin{equation}
    \Psi_{ab}(x) =(\frac{1}{2\pi})^\frac{3}{2}\sum_{\lambda}\int d^3k \, \left\{ (-)^{2B}u_{ab}(\boldsymbol{k},\lambda)a^{c\dagger}(\boldsymbol{k},\lambda) e^{-ikx} +  v_{ab}(\boldsymbol{k},\lambda) a(\boldsymbol{k},\lambda)e^{ikx  } \right\}
\end{equation}
以及系数函数的选择变为：
\begin{subequations}
    \begin{align}
        &\sum_{\bar{\lambda}}u_{\bar{a}\bar{b}}(\Lambda \boldsymbol{k},\bar{\lambda})D_{\bar{\lambda}\lambda }^{j_n*}\left\{W(\Lambda,p)\right\}=\sqrt{\frac{p^0}{(\Lambda p)^0}}\sum_{\bar{a}\bar{b}}\mathscr{D}_{\bar{a}\bar{b};ab}(\Lambda)u_{ab}(\boldsymbol{k},\lambda)
        \\
        &\sum_{\bar{\lambda}}v_{\bar{a}\bar{b}}(\Lambda \boldsymbol{k},\bar{\lambda})D_{\bar{\lambda}\lambda }^{j_n}\left\{W(\Lambda,p)\right\}=\sqrt{\frac{p^0}{(\Lambda p)^0}}\sum_{\bar{a}\bar{b}}\mathscr{D}_{\bar{a}\bar{b};ab}(\Lambda)v_{ab}(\boldsymbol{k},\lambda)
    \end{align}
\end{subequations}
首先确定系数的选择；考虑剩下的两组Poincar\'e变换，依次考察$\operatorname{Boost}$和$\operatorname{Rotation}$；
\paragraph*{Boost}
\begin{itemize}
    \item 令 $k^\mu = (\kappa,0,0,\kappa)$为标准动量系；
    \item 令 $\Lambda=L(q)$,表示从标准动量系到任意系的标准Lorentz变换，即:$\Lambda k=q^\mu=(|\boldsymbol{q}|,\boldsymbol{q})$
\end{itemize}
那么 
\[\begin{aligned}
    \begin{aligned}
    &W(\Lambda,k)=L^{-1}(\Lambda k)\Lambda L(k)\\
    &=L^{-1}(q) L(q)=1
    \end{aligned}\to\sum_{\bar{\lambda}}D_{\lambda \bar{\lambda}}^{j_n}\!\left\{W^{-1}(\Lambda,k)\right\} =1
\end{aligned}\]
所以系数函数选择变为：
\begin{subequations}
    \begin{align}
        &\sum_{\bar{\lambda}}u_{\bar{a}\bar{b}}(\boldsymbol{q},\bar{\lambda})\delta_{\bar{\lambda}\lambda}=\sqrt{\frac{\kappa}{|\boldsymbol{q}|}}\sum_{\bar{a}\bar{b}}\mathscr{D}_{\bar{a}\bar{b};ab}(\Lambda)u_{ab}(\boldsymbol{k},\lambda)
        \\
        &\sum_{\bar{\lambda}}v_{\bar{a}\bar{b}}(\boldsymbol{q},\bar{\lambda})\delta_{\bar{\lambda}\lambda}=\sqrt{\frac{\kappa}{|\boldsymbol{q}|}}\sum_{\bar{a}\bar{b}}\mathscr{D}_{\bar{a}\bar{b};ab}(\Lambda)v_{ab}(\boldsymbol{k},\lambda)
    \end{align}
\end{subequations}
因此，任意动量处的系数完全由标准动量处的系数 $u(\boldsymbol{k},\lambda)$ 和 $v(\boldsymbol{k},\lambda)$ 决定。
\paragraph*{Rotation}
前文中，已经解决了$ISO(2)$小群带来的约束问题，现在开始讨论旋转条件。
\begin{itemize}
    \item 令 $k^\mu = (\kappa,0,0,\kappa)$为标准动量系；
    \item 对$ISO(2) $的讨论中；Winger的小群的群元只有$J_3$是有效的；所以选择$\Lambda=J_3$也就是在标准动量下保持$k^\mu = (\kappa,0,0,\kappa)$不变的旋转；
\end{itemize}
那么
\[W(\theta,\alpha,\beta)=S(\alpha,\beta)R_3(\theta)=J_3\]
由于$\boldmath{J}=\boldsymbol{\mathcal{A}}+\boldsymbol{\mathcal{B}}$，$Winger$的转动表征为总的自旋$\boldsymbol{J}$,以及(\refeq{massless_lorentz_transf})和上述已经消除了非物理的情况，所以总自旋的表示矩阵为：
\[
D_{\lambda\bar\lambda}^{(j)}(J_3^{-1})=\left[\exp(+iJ_3)\right]_{\lambda\bar\lambda}
\]
场的自旋拆为$\boldsymbol{\mathcal{A}}+\boldsymbol{\mathcal{B}}$,表示矩阵 $\mathscr{D}(J_3^{-1})$ 也可以分解为 $A$ 和 $B$ 两部分的张量积：
\[
\mathscr{D}(J_3^{-1}) = \exp\left( +i \theta J_3 \right) = \exp\left( +i \theta (\mathcal{A}_3 + \mathcal{B}_3) \right) = \exp\left( +i \theta \mathcal{A}_3 \right) \otimes \exp\left( +i \theta \mathcal{B}_3 \right)
\]
在这种分解下交织方程变为：  
\begin{subequations}\label{massless_rotation}
    \begin{align}
        &\sum_{\bar\lambda} u_{ab}(\boldsymbol{k},\bar\lambda)\, e^{-i \theta J_3} \delta_{\lambda\bar\lambda} = \sum_{\bar{a}\bar{b}} \left[ e^{+i \theta \mathcal{A}_3} \right]_{a\bar{a}} \left[ e^{+i \theta \mathcal{B}_3} \right]_{b\bar{b}} u_{\bar{a}\bar{b}}(\boldsymbol{k},\lambda)\\
        &\sum_{\bar\lambda} v_{ab}(\boldsymbol{k},\bar\lambda)\,e^{+i \theta J_3} \delta_{\lambda\bar\lambda} = \sum_{\bar{a}\bar{b}} \left[ e^{+i \theta \mathcal{A}_3} \right]_{a\bar{a}} \left[ e^{+i \theta \mathcal{B}_3} \right]_{b\bar{b}} v_{\bar{a}\bar{b}}(\boldsymbol{k},\lambda)
    \end{align}
\end{subequations}
按照5.3的逻辑，以及(\refeq{helcity_o})就是选择$a = -A$ 和 $b = +B$,那么系数函数就是
\begin{equation}\label{massless_c}
    v_{ab}(\boldsymbol{k},\lambda) =\frac{1}{\sqrt{2\kappa}} \langle A,a;B,b \,|\, j, \lambda \rangle=\frac{1}{\sqrt{2\kappa}} \langle -A; +B \,|\, j, \lambda \rangle  
\end{equation}
对其做一标准boost $\mathscr{L}(k)$，得到任意动量处的系数函数：
\begin{equation}
    \begin{aligned}
    v_{ab}(\boldsymbol{q},\lambda) = &\sqrt{\frac{\kappa}{|\boldsymbol{q}|}} \sum_{\bar{a}\bar{b}} \left[ R(\hat{\boldsymbol{k}}) \right]_{a\bar{a}} \left[ R(\hat{\boldsymbol{k}}) \right]_{b\bar{b}} v_{\bar{a}\bar{b}}(\boldsymbol{k},\lambda)\\
        &= \sqrt{\frac{1}{2|\boldsymbol{q}|}} \sum_{\bar{a}\bar{b}} \left[ R(\hat{\boldsymbol{k}}) \right]_{a\bar{a}} \left[ R(\hat{\boldsymbol{k}}) \right]_{b\bar{b}}  \langle -A; +B \,|\, j, \lambda \rangle\\
    \end{aligned}
\end{equation}\label{massless_boost_u_k}
其中 $R(\hat{q})$ 是将 $\hat{z}$ 旋转到 $\hat{q}$ 方向的旋转矩阵,
以及反粒子的系数\footnote{准确的说和我们熟悉的对偶场是一个说法}
\begin{equation}
    u_{ab}(\boldsymbol{q},\lambda) =v_{ab}(\boldsymbol{q},\lambda)^*
\end{equation}



\topictitle{零质量因果场的一般构造}
现在表述零质量粒子因果场的完整构造逻辑,为后续各自旋的具体实现 (spin-1, spin-2, $\ldots$) 提供统一框架。
\paragraph*{选择表示。}
对于spin$=j$ 的零质量粒子, Helcity是$\lambda=\pm j$。
 $(A,B)$ 表示以及物理上的约束，所以螺旋度是$\lambda=B-A$。
因此描述 $\lambda=+j$ 需要满足 $B-A=j$ 的表示,
描述 $\lambda=-j$ 需要 $A-B=j$ 的表示。
最小的选择是
\begin{equation}
    \lambda = +j:\; (A,\,A+j), \qquad \lambda = -j:\; (A+j,\,A)
    \label{eq:massless-rep-choice}
\end{equation}
其中 $A$ 可以取 $0, \frac{1}{2}, 1, \ldots$\,\footnote{%
不同的 $A$ 给出描述同一物理粒子的不同场。
例如光子 ($j=1$) 可以用 $(1,0)\oplus(0,1)$ (场强张量, $A=0$)
或 $(\frac{1}{2},\frac{1}{2})$ (矢量势, $A=\frac{1}{2}$, $B=\frac{1}{2}$,
但此时 $B-A=0\neq j$, 需要额外约束去掉 $j=0$ 分量)。
最``经济"的选择是 $A=0$, 即 $(0,j)\oplus(j,0)$,
此时表示空间维度最小且不包含多余的低自旋分量。}。
由 3.2 节关于空间反演的讨论, 宇称变换 $\mathcal{P}:(A,B)\to(B,A)$,
因此为保持宇称对称性, 必须取直和
\begin{equation}(A,B) \oplus (B,A), \qquad B-A = j
    \label{eq:massless-parity-rep}
\end{equation}
最简单的选择 $A=0$ 给出
\begin{equation}
    (0,j) \oplus (j,0)
    \label{eq:minimal-rep}
\end{equation}
表示空间维度为 $2(2j+1)$。
\paragraph*{系数函数：CG 系数的退化}
由 一般结果 (\refeq{massless_c}), 标准动量处的系数函数为
\begin{equation}
    v_{ab}(\boldsymbol{k},\lambda) = \frac{1}{\sqrt{2\kappa}}
    \langle A,a;\,B,b\,|\,j,\lambda\rangle
\end{equation}
ISO(2) 约束 (\ref{eq:ISO2-constraint-general}) 将其简化为
\begin{equation}
    v_{-A,+B}(\boldsymbol{k},\lambda) = \frac{1}{\sqrt{2\kappa}}
    \langle A,-A;\,B,+B\,|\,j,\lambda\rangle, \qquad
    \text{其余分量} = 0
    \label{eq:v-massless-general}
\end{equation}
CG 系数 $\langle A,-A;\,B,+B\,|\,j,B-A\rangle$ 在 $j=A+B$ 时等于 1\,\footnote{%
这是 CG 系数的``拉伸态" (stretched state) 性质:
$\langle j_1,m_1;\,j_2,m_2\,|\,j_1+j_2,\,m_1+m_2\rangle = 1$
当 $m_1=-j_1$, $m_2=+j_2$ (或其排列) 时。},
因此对于最经济的表示 (\ref{eq:minimal-rep}) ($A=0$ 或 $B=0$, $j=A+B$):
\begin{equation}
    v_{-A,+B}(\boldsymbol{k},\lambda) = \frac{1}{\sqrt{2\kappa}}
\end{equation}
系数函数在标准动量处退化为一个纯数——
所有的张量/旋量结构完全由球面基到物理分量的基变换矩阵承载。
\paragraph*{标准 boost 到任意动量。}
由 (\refeq{massless_boost_u_k}), 任意动量处的系数函数为
\begin{equation}
    v_{ab}(\boldsymbol{q},\lambda) = \sqrt{\frac{1}{2|\boldsymbol{q}|}}
    \sum_{\bar{a}\bar{b}}
    \bigl[R(\hat{q})\bigr]_{a\bar{a}}\,
    \bigl[R(\hat{q})\bigr]_{b\bar{b}}\,
    \langle A,-A;\,B,+B\,|\,j,\lambda\rangle
    \label{eq:v-boost-general}
\end{equation}
由 (3.101), 反粒子 (或产生部分) 的系数为
$u_{ab}(\mathbf{q},\lambda) = v_{ab}(\mathbf{q},\lambda)^*$。将以上结果代入一般场展开 (3.94), 对于最简单的表示 $(0,j)\oplus(j,0)$,
因果场为
\begin{equation}
    \Psi_{ab}(x) = \frac{1}{(2\pi)^{3/2}} \sum_{\lambda=\pm j}
    \int d^3k
    \left[(-1)^{2B}\,u_{ab}(\boldsymbol{q},\lambda)\,a^{c\dagger}(\boldsymbol{q},\lambda)\,e^{-iq\cdot x}
    + v_{ab}(\boldsymbol{q},\lambda)\,a(\boldsymbol{q},\lambda)\,e^{iq\cdot x}\right]
    \label{eq:massless-field-general}
\end{equation}
其中 $(-1)^{2B}$ 由 3.3 节微观因果性分析确定 (参考 (3.84)).\textbf{自旋-统计与微观因果性。}
3.3 节的分析对零质量情况完全适用:
整数自旋 ($j=1,2,\ldots$) 场满足对易关系 (玻色子),
半整数自旋 ($j=\frac{3}{2},\frac{5}{2},\ldots$) 场满足反对易关系 (费米子)。
微观因果性
$[\Psi_{ab}(x),\,\Psi^\dagger_{cd}(y)]_\mp = 0$ ($(x-y)^2<0$)
自动保证。
